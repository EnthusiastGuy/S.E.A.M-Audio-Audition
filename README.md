# S.E.A.M-Audio-Audition
S.E.A.M. (Segmented Evaluation &amp; Audition Module) is a single-file, local web app for dynamic music previews. Designed for game devs, it parses branching filenames (e.g., 1 -> 2, 4) to simulate real-time transitions, seamless loops, and multi-track crossfades. Features a flat UI, Web Audio API precision, and playback speed control.
