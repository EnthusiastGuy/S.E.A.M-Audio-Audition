Getting Started

Click Select Folder on the welcome screen and choose the root folder of your sample pack — the one that contains the mp3, ogg, and wav sub-directories. The app reads all files in-browser; nothing is uploaded anywhere.

Song List (Tabs)

Three tabs — MP3 / OGG / WAV — show songs discovered in each format folder. Each row shows the song number, name, total duration, part count, and transport controls.

  • Drag rows up or down to reorder the playlist.
  • A row shown in amber means the main (full) song file is missing; only loop parts are available. Hover the name for a tip.
  • If neither the main file nor any parts exist, the song is hidden.

Playing a Song

Press the ▶ Play button on any row. The button changes to ⏸ Pause + ■ Stop. The seek bar expands below the row. Multiple songs can play simultaneously — useful for auditioning layers.

When a song finishes, the next one in the playlist starts automatically. After the last song the queue wraps back to the top.

Seek Bar

  • Coloured bricks represent each part in proportion to its duration.
  • Click any brick to jump to that part's start.
  • Drag the white handle anywhere to seek.
  • The time label above the handle shows total played time, including repeated loops.
  • The percentage on the right shows position relative to the original (unlooped) duration.

Loop Controls

Loopable parts (those whose filename contains -> and list themselves as a next destination) show a Loop button centred below their brick. Click it to open the loop dropdown:

  • Continue (once) — plays through without looping.
  • 2× … 10× — loops that part N times before moving on.
  • Forever ∞ — loops indefinitely until you manually seek past that part.

The button label updates in real-time to show loop progress, e.g. Loop 2/3.

Part Sheet / Timeline Editor

Click the numbered Parts badge in the Parts column to expand the part sheet. You will see:

  • A list of all available files for that song at the top.
  • The current timeline (bricks) at the bottom, which you can edit by drag-and-drop.
  • Drag a brick in the timeline left/right to reposition it.
  • Drag a file from the list above onto the timeline to insert it between existing bricks.
  • Hovering a file highlights its instances in the timeline (and vice-versa).

Crossfade

The Crossfade slider at the bottom (0 – 5000 ms) controls the overlap between consecutive playlist songs. When the current song has fewer than N ms remaining, the next song begins quietly and fades in while the current one fades out.

Playback Speed Knob

The rotary knob controls playback rate for all active audio. Turn clockwise to increase speed, counter-clockwise to slow, reverse, or stop.

  • North-East: 100% (default)
  • South-East: 200% (max forward)
  • North: 0% (stopped)
  • North-West: −100% (reverse)
  • South-West: −200%
  • South: −250% (max reverse)

The knob snaps in 5% increments with a click sound. It cannot jump directly from South-East to South — you must turn it counter-clockwise through the North positions.

Session Persistence

All playlist order, loop settings, crossfade value, speed, and part arrangements are saved to localStorage automatically. Reopen the page (with the same folder) and find everything exactly as you left it.

File Naming Convention

Parts follow the pattern: Song Name N -> A, B, C.ext where N is the part number and A, B, C are the part numbers that may follow it. The main (full) song file has just the song name with no part number.
